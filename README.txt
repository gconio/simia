SimIA cumulative upload package

Included
- activation_monitor.html
- instructor.html (redirect to activation_monitor.html)
- activations_admin.html
- observer.html
- functions/api/scenario/observer_feed.js
- README.txt

What this package does
1) Instructor is re-framed as Activation Monitor
2) Activations Setup keeps a minimal live planning preview
3) Observer gets a dedicated read-only Exercise Overview
4) Existing player page can remain as Situation Workspace
5) Menus are aligned around monitor / setup / observer

Install
1. Add:
   functions/api/scenario/observer_feed.js
   activation_monitor.html
   observer.html
2. Replace:
   activations_admin.html
   instructor.html
3. Commit + Push
