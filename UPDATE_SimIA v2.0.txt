SimIA v2.0 — Scenario Engine Foundation (Hybrid)

What this package adds
- Scenario data model:
  - scenarios
  - scenario_phases
  - scenario_injects
  - scenario_actions
- APIs:
  - /api/scenario/manage
  - /api/scenario/phases
  - /api/scenario/injects
  - /api/scenario/actions
- Standalone admin page:
  - /scenario_admin.html

How to install
1) Copy files into repo preserving folders
2) Run D1 SQL from migrations/050_scenario_engine_foundation.sql
3) Commit + Push

How to use
1) Open /scenario_admin.html
2) Create a scenario and set it current
3) Add phases
4) Add injects
5) Inspect white cell log

Notes
- This is the foundation layer for Hybrid simulation
- It does not yet auto-release scheduled injects
- Next step will add:
  - release now
  - postpone
  - cancel
  - scheduled queue
  - white cell override actions
